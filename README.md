# Arduino-Workshop-42-Chips

This repository contains my personal answers and solutions to the **42 Chips Arduino Discovery Workshop** (*Atelier Découverte Arduino*).

## About the Workshop

The workshop is an introductory hands-on session designed to help beginners get started with Arduino programming and electronics. It covers fundamental concepts such as digital I/O, analog signals, sensors, LEDs, and basic circuit design.

The subject (in French) is available here:
👉 [Atelier Découverte Arduino – 42Chips Notion](https://42chips.notion.site/Atelier-d-couverte-Arduino-31ed6f77142c8067be1fd4f85c1bfc9a)

The workshop was produced by **[42Chips](https://42chips.fr/)** and **Labelec**.

## Repository Contents

Each file or folder in this repository corresponds to an exercise or project from the workshop subject.
